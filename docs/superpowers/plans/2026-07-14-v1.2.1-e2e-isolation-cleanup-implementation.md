# V1.2.1 E2E Isolation and Cleanup Implementation Plan

> **中文说明：** 本实施计划将浏览器门禁数据迁移到 `e2e` 租户，并清理每次成功运行创建的全部测试资源。

> **For agentic workers:** REQUIRED SUB-SKILL: Use $superpower-subagents (recommended) or $superpower-executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking via update_plan.

**Goal:** Move real-browser gate data into tenant `e2e` and remove every successful-run test resource.

**Architecture:** A single `OPS_ADMIN` endpoint physically deletes only lower-case `e2e_*` accounts whose tenant is exactly `e2e`. Browser E2E creates a temporary E2E administrator, switches sessions, and deletes resources through the same cookie and CSRF boundary as the console.

**Tech Stack:** Spring Boot, JPA, React/Vitest, Playwright, Docker Compose.

@Transactional
public String deleteE2e(String username) {
    UserAccount user = requireUser(username);
    if (!user.getUsername().startsWith("e2e_") || !"e2e".equals(user.getTenantId())) {
        throw new IllegalArgumentException("only e2e_* accounts in the e2e tenant can be deleted");
    }
    userAccountRepository.delete(user);
    return user.getUsername();
}

- [x] **Step 2: Implement and verify GREEN lifecycle helper**

```ts
export interface E2eRunResources {
  runId: string
  gateUsername: string
  accountUsername: string
  kbName: string
  kbId?: string
}

export async function finalizeE2eRun(completed: boolean, resources: E2eRunResources,
  cleanup: () => Promise<void>, attach: (name: string, body: string) => void | Promise<void>): Promise<void> {
  if (!completed) {
    await attach('e2e-evidence', JSON.stringify(resources, null, 2))
    return
  }
  try {
    await cleanup()
  } catch (error) {
    await attach('e2e-evidence', JSON.stringify({
      ...resources,
      cleanupError: error instanceof Error ? error.message : String(error),
    }, null, 2))
    throw error
  }
}
```

Run `npm test -- src/test/e2eLifecycle.test.ts`; product-failure, asynchronous evidence attachment, success-cleanup, and cleanup-failure cases pass.
