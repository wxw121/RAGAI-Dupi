> **中文说明：** 本实施计划安全重建 V1.5 合并到 `main` 的过程，清除提交中的冲突标记，同时保留 V1.4.2 与 V1.5 的治理、恢复和检索质量能力。

**目标：**在保留V1.4.2治理/恢复行为和V1.5检索质量行为的同时，重新创建没有提交冲突标记的V1.5到main合并。

**技术栈：** Java 17/Spring Boot/Maven， Python/pytest, React/TypeScript/Vitest/Vite, Git。
