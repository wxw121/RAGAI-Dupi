# V1.3 P0 RAG质量门和检索实施计划

<!-- language-switch -->
[English](../../../en/superpowers/plans/2026-07-14-v1.3-p0-rag-quality-retrieval-implementation.md)





> **中文说明：** 本实施计划增加可阻断发布的 RAG 质量基线，并把混合检索从请求时内存 BM25 迁移到带版本的 Milvus 稀疏检索配置。

>对于代理工作者：使用superpower- execution -plans逐个任务执行计划。步骤使用update_plan的复选框语法。

**目标：**添加一个可阻塞的RAG质量基线，并将混合检索从请求时内存BM25迁移到版本化的Milvus稀疏配置文件。

**架构：**评估运行持久不变的度量、案例指纹、比较状态和概要快照。每kb策略和显式基线产生门结果。新版本的稀疏集合在切换前进行回填和双重写入。

**技术栈：** Spring Boot/JPA/Flyway， Milvus, Python/FastAPI, React/Vite/Vitest，剧作家，PowerShell。
