# V1.4.0可验证恢复实施计划

<!-- language-switch -->
[English](../../../en/superpowers/plans/2026-07-15-v1.4.0-verifiable-recovery-implementation.md)





> **中文说明：** 本实施计划增加租户隔离、校验和验证的知识库归档，并支持幂等恢复到新的隐藏知识库。

b> **对于代理工作者：**需要的子技能：使用$superpower-subagents（推荐）或$superpower- execution -plans逐个任务地执行此计划。步骤使用复选框（' -[]'）语法通过update_plan进行跟踪。

**目标：**添加租户隔离，校验和验证知识库档案和幂等恢复到一个新的隐藏知识库。

架构：恢复元数据和项目进度存储在PostgreSQL中，存档有效负载存储在专用的私有MinIO桶中的密封前缀下，密集/稀疏向量通过集中的Milvus快照适配器流式传输。后台执行器推进显式归档和恢复状态机；稳定的项键和确定的目标标识符使得跨进程或依赖失败的重试是安全的。

**技术栈：** Java 17， Spring Boot/JPA/Flyway, MinIO Java SDK, Milvus Java SDK, PostgreSQL, React/Vite/Vitest，剧作家，PowerShell。
